##### configure my vim

```
1. git clone http://github.com/VundleVim/Vundle.vim.git ~/.vim/bundle/vundle
2. cp the vimrc file to home director.
3. start vim, input ":BundleInstall", you only need wait the "Done" tipes
4. sudo apt-get install llvm-3.9 clang-3.9 libclang-3.9-dev libboost-all-dev
5. compile YCM
	4.1 cd ~/.vim/bundle/YouCompleteMe
	4.2 ./install.sh --clang-completer
```
